Quem desenvolveu o sistema?

Esse sistema foi desenvolvido pelo estudante de análise e desenvolvimento de sistemas Heliel Souza Lima, na instituição de ensino COTUCA, no final de 2023.


O projeto foi idealizado através de um Trabalho de Conclusão de Curso (TCC), onde no qual, a idéia surgiu por conta de seu ex-colega de sala, Anderson.


Anderson, dias depois, saiu do curso, onde todo o trabalho de desenvolvimento foi atribuído ao Heliel. A idéia se amadureceu cada vez mais, onde foram incluídas novas funcionalidades até chegar ao produto final.

Como funciona esse sistema?

Esse sistema foi criado para ser usado em um controle de laboratório que realiza exames de sangue, com diversas funcionalidades incluídas:

  • Geração de laudos de hemograma através de imagens microscópicas com um modelo de IA treinado;
  • Controle e cadastro de usuários (Pacientes, Técnicos, Recepcionistas e Administradores);
  • Permissões específicas para cada tipo de usuário;
  • Controle de laudos criados por técnicos associando à um paciente;
  • Login e meu perfil, com todas as movimentações e dados pessoais;

    Qual o objetivo deste sistema?

    O maior objetivo desse sistema é aprimorar a fase introdutória de uma realização de exame sanguíneo, retornando em questões de segundos o resultado em forma de hemograma de uma análise de uma imagem de sangue microscópica.


    Além disso, com base nos tipos de usuários que podem ser criados, as permissões de acesso e funções dentro do sistema, junto ao design simples e limpo, trazem uma dinânica maior ao controle de laudos, pacientes, atividades internas e muito mais

    Tecnologias utilizadas

    Django: O projeto majoritariamente foi desenvolvido neste framework, que é escrito em python no qual a criação de aplicações web tanto no lado do servidor quanto no lado do cliente. A escolha foi para concatenar com a IA que utilizei, também escrita em python.


    YOLO (You Only Look Once): Algoritmo de detecção de objetos de interesse em cena. Esse algoritmo foi usado para treinar um DataSet de mais de 400 imagens sanguíneas no Google Colab que tiveram bounding-boxes criados manualmente. o YOLO utiliza a arquitetura de rede neural conhecida como CNN (Convolutional Neural Network) para realizar a detecção de objetos. As CNNs são especialmente adequadas para tarefas de visão computacional, pois são capazes de aprender representações hierárquicas de características em imagens, o que as torna muito eficazes na detecção e classificação de objetos.


    Darknet: Framework utilizado para treinar o modelo utilizando YOLO com algoritmo.


    JavaScript, Jquery, AJAX: Utilizadas para aperfeiçoar a interação com o usuário.


    HTML e CSS: Usadas para estrutração e estilização da página.
