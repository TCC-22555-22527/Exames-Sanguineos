No sistema existem 4 tipos de usuários que podem ser criados e utilizados:


Pacientes

O que é liberado

  • Consulta de laudos que foram criados com seu CPF ou nome de cadastro
  • Visualização de seus dados pessoais
  • Alteração de senha do perfil

    • O que é bloqueado

  • Visualização de qualquer outro perfil, desde dados, laudos, etc
  • Tudo aquilo que os usuários abaixo tem permissão
    • Recepcionistas

    O que é liberado

  • Cadastro de usuários pacientes
  • Alterações de dados pessoais de pacientes (mediante a solicitação do mesmo, tal como endereço, por exemplo)
  • Consulta de laudos já existentes no sistema
  • Visualização dos detalhes de laudos criados no sistema
  • Pesquisa de pacientes
  • Visualização geral de qualquer perfil do tipo 'paciente'
  • Visualização de seus dados pessoais
  • Alteração de senha do seu perfil
  • Visualização de pacientes cadastrados pelo seu perfil

    • O que é bloqueado

  • Cadastro de usuários técnicos e administradores
  • Envio de imagens para geração de laudo de hemograma
  • Alteração de senhas de qualquer outro perfil
    • Técnicos

    O que é liberado

  • Envio de imagens para serem analisadas
  • Consulta de laudos já existentes no sistema
  • Visualização dos detalhes de laudos criados no sistema
  • Pesquisa de pacientes
  • Visualização geral de qualquer perfil do tipo 'paciente'
  • Alteração de senha próprio
  • Visualização de seus dados pessoais
  • Visualização de laudos gerados pelo seu perfil

    • O que é bloqueado

  • Cadastro de qualquer tipo de usuário
  • Alteração de dados de pacientes
  • Alteração de senhas de qualquer outro perfil
    • Administradores

    O que é liberado

  • Todas as funcionalidades do sistema
